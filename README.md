# Memory App

The Memory App is a browser based Memory Game. It is built
with Typescript, SCCS and Vite.

---
## Features

- 4 selectable themes
- 2 chooseable players
- 3 board sizes
- Live score tracker
- Animated cards
- Animated score screen

## How to install

1. **Fork the Repository**
2. **Install required Dependencies**
    - Typescript 6.0.3 or higher
    - SASS 1.101.0 or higher
    - Vite 8.0.16 or higher
3. **Run ```npm run dev``` for preview**
4. **Run ```npm run build``` for FTP server files**
5. **Copy content in "dist" folder and insert it on your FTP server**

## How to play

1. Settings
   - Select your desired settings
  
  | Setting | Options |
  |---|---|
  | **Game Theme** | Code Vibe, Gaming, DA-Projects, Food |
  | **Starting Player** | Blue, Orange |
  | **Board Size** | 16, 24, 36 cards |
  
2. Click on cards to flip them
  - Flip two matching cards to obtain a point and continue playing
  - Flip two non-matching cards and you will not recieve a point, the next player continues

3. How to win?
   - In order to win you will have to obtain more points than the other player
   - If both players have the same amount of points the game will end in a draw
  
4. Replay
   - After a match, you have the option to go back to the home menu, to start a new game
   - You can also leave your current match that is still ongoing, taking you back to the settings
