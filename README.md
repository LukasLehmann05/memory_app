# Memory App

The Memory App is a browser based Memory Game. It is built
with Typescript, SCCS and Vite.

---
## Features

- 4 selectable themes
- 2 chooseable players
- 3 board sizes
- Live score tracker
- Animated cards
- Animated score screen

## How to install

1. **Fork the Repository**
2. **Install required Dependencies**
  - Typescript 6.0.3 or higher
  - SASS 1.101.0 or higher
  - Vite 8.0.16 or higher
3. **Run "npm run dev" for preview**
4. **Run "npm run build" for FTP server files
5. **Copy content in "dist" folder and insert it on your FTP server

## How to play

1. Select desired settings in the settings page and press play
2. Click on cards to flip them
3. Reveal 2 cards with the same image to obtain a point
4. Once the entire field is uncovered the game will end
5. A screen will pop up revealing points and winner
